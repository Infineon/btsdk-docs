WICED® BT SDK Documentation v2.1
ModusToolbox contains the WICED BT SDK, which supports multiple WICED Bluetooth devices. WICED APIs are specific to each chip family of WICED devices, and are documented separately per family with Doxygen generated API documentation. The links below contain Datasheets, Kit Guides, and Doxygen API documentation for each supported chip family.
