Detailed Description

Macro Definition Documentation

#define BUFFER_SIZE   120

Defines an audio codec interface.

Applications that use this interface can plug in different audio codec with no impact. Audio codec that comply with this interface can be used by different applications. The BCM remote application uses this interface to control audio codec activities.
