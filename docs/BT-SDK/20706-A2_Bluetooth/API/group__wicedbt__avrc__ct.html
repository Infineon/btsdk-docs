Detailed Description

WICED Bluetooth AVRC Remote Control CT Functions.

Enumeration Type Documentation

enum wiced_bt_avrc_ct_connection_state_t

AVRC remote control connection state.

Enumerator
REMOTE_CONTROL_DISCONNECTED 

Remote control is disconnected from peer.

REMOTE_CONTROL_CONNECTED 

Remote control is connected to peer.

REMOTE_CONTROL_INITIALIZED 

Remote control is idle (not connected to any peer)

enum wiced_bt_avrc_ct_features_t

AVRC remote control feature mask.

Enumerator
REMOTE_CONTROL_FEATURE_TARGET 

Select to support the features of a Target.

REMOTE_CONTROL_FEATURE_CONTROLLER 

Select to support the features of a Controller.
