Detailed Description

Enumeration Type Documentation

anonymous enum

Defines an Rtc driver.

20730/20733 support 48 bits RTC timer from 32kHz crystal oscillator.

Usage:

(1) use Rtc->getInstance() to get the RTC driver instance first.

(2) use Rtc->setRTCTime() to set the current calender time

(3) user Rtc->getRTCTime() to retrieve the current calender time.

(4) Rtrc->ctime(), will convert the RTC_time to ascii user friendly string.
