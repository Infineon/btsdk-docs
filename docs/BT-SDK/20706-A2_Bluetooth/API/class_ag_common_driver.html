virtual void getMotion ( INT16 *  x,
INT16 *  y,
BYTE  maxSensorReads 
)
virtual

This function should check for motion and add any detected motion to the x/y variables passed in.

Note that the function must add new motion to the x/y variables; it should not replace what is in there or motion may be lost. Some sensors have the ability to accumulate more motion than they can report in a single poll and require multiple polls to get all the accumulated motion. In such cases the driver is allowed to poll the sensor multiple times, up to a maximum of maxSensorReads. An example of such a sensor is the AG3040, which can accumulate more than a bytes worth of motion but reports it only a byte at a time (-128 to 127). By limiting the number of polls an application can bound the time it takes to generate a report.

Parameters
*xpointer to variable to which detected x motion should be added.
*ypointer to variable to which detected y motion should be added.
maxSensorReadsmaximum number of times the motion sensor HW can be polled. Must be >= 1 or the result is undefined.
