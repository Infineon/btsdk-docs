void *  pOut,
int  pcm_count

Volume control - Ramp up/down the PCM samples during mute/un-mute.

This API is called by the codec library after the decoding is done to allow FW to ramp the audio data when mute/un-mute is requested by the application.

Parameters
pInpointer to the pcm input buffer
pInpointer to the pcm output buffer
