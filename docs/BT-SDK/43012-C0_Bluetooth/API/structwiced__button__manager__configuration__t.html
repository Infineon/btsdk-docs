wiced_bool_t continuous_hold_detect

Flag to enable button holding detection.


If this is set to true, the BUTTON_VERY_LONG_DURATION_EVENT is disabled and a BUTTON_HOLDING_EVENT will be triggered if a button is pressed over twice the long_hold_duration.
The BUTTON_HOLDING_EVENT will continuous be sent every long_hold_duration after the first BUTTON_HOLDING_EVENT event been sent.
