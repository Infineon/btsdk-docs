Enumeration Type Documentation

anonymous enum

Defines a MIA driver.

Applications use this driver to obtain the status from and control the behavior of the MIA driver.Mia constants are defined below

Enumerator
MIA_P2_OPSR_SEL_PORT_2_6_BIT_MASK 

Mask for P2_6(GPIO_22). 00 in the mask will be normal P2_6 or GPIO_22.

MIA_P2_OPSR_SEL_PORT_2_6_IS_KSO14 

This value in the above mask would set KSO14 output.

MIA_OPSR_SEL0_PORT_2_7_BIT_MASK 

Mask for P2_7(GPIO_23). 00 value in the mask would set normal GPIO_23.

MIA_OPSR_SEL0_PORT_2_7_IS_KSO15 

This value in the above mask would set KSO15 output.

MIA_OPSR_SEL0_PORT_3_0_BIT_MASK 

Mask for P3_0(GPIO_24). 00 value in the mask would set normal GPIO_24.

MIA_OPSR_SEL0_PORT_3_0_IS_KSO16 

This value in the above mask would set KSO16 output.

MIA_OPSR_SEL0_PORT_3_1_BIT_MASK 

Mask for P3_1. 0 value in the mask would set normal GPIO_25.

MIA_OPSR_SEL0_PORT_3_1_IS_KSO17 

This value in the above mask would set KSO17 output.

MIA_OPSR_SEL0_PORT_3_2_BIT_MASK 

Mask for P3_2. 0 value in the mask would set normal GPIO_26.

MIA_OPSR_SEL0_PORT_3_2_IS_KSO18 

This value in the above mask would set KSO18 output.

MIA_OPSR_SEL1_PORT_3_3_BIT_MASK 

Mask for P3_3. 0 value in the mask would set normal GPIO_27.

MIA_OPSR_SEL1_PORT_3_3_IS_KSO19 

This value in the above mask would set KSO19 output.

enum MiaTimedWakeRefClock

Reference clock for timed wake from deep sleep.

Enumerator
HID_OFF_TIMED_WAKE_CLK_SRC_32KHZ 

Use the 32 KHz clock for the RTC to give a range of ~64 ms - ~36 Hrs.

HID_OFF_TIMED_WAKE_CLK_SRC_128KHZ 

Use the 128 KHz clock for the RTC to give a range of ~256 ms - ~144 Hrs.

BOOL32 mia_getLhlInterruptState ( void  )

Get the LHL interrupt state.

This state may not be the actual hardware LHL interrupt setting, since the interrupt context interrupt handler will disable the LHL interrupt without updating the state variable, which is a good thing since it makes it easy to restore the correct hardware setting from this state variable when done handling the serialized interrupt by calling restoreLhlInterruptSetting().

Returns
  • true if lhl interrupts should be enabled.
  • false if lhl interrupts should be disabled
BOOL32 mia_getMiaInterruptState ( void  )

Get the MIA interrupt state.

This state may not be the actual hardware MIA interrupt setting, since the interrupt context interrupt handler will disable the MIA interrupt without updating the state variable, which is a good thing since it makes it easy to restore the correct hardware setting from this state variable when done handling the serialized interrupt by calling mia_restoreMiaInterruptState().

Returns
  • true if MIA interrupts should be enabled.
  • false if MIA interrupts should be disabled

void mia_pollHardware ( void  )

Poll all HW blocks on the MIA domain if one or more events are detected.

Keyscan and Quadrature HW blocks are part of the MIA power domain. Events from either or both are processed by freezing the MIA clock, pulling events out of the HW block and then unfreezing the MIA clock.
