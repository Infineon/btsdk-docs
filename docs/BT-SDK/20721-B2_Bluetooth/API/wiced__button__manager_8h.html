Button manager defines generic interface for button events and button type configurations. It exposes interface to configure platform button events (like click,long press) with user configurable timing.

enum button_manager_event_t

Button Manager events.

Enumerator
BUTTON_CLICK_EVENT 

A click is a combination of press and release button events.

Typically ~ < 200 ms.

BUTTON_SHORT_DURATION_EVENT 

Short duration click event.

BUTTON_MEDIUM_DURATION_EVENT 

Medium duration click event.

BUTTON_LONG_DURATION_EVENT 

Long duration click event.

BUTTON_VERY_LONG_DURATION_EVENT 

Very long duration click event.

BUTTON_DOUBLE_CLICK_EVENT 

A double click is a combination of two single clicks with some delay between them.

BUTTON_HOLDING_EVENT 

Button is holding over at least two long duration.
