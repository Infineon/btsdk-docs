Detailed Description

Enumeration Type Documentation

anonymous enum

Defines the BCM standard keyscan driver.

The BCM standard keyscan driver provides the status and control for the keyscan driver. It provides the keyEvents to the interface, maintains the queue behind it. It also supports keyscanning turning on or off.

Enumerator
EVENT_NONE 

Keycode value if no key is pressed.
