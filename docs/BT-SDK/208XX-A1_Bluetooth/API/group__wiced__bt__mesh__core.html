Detailed Description

Mesh Core library of the WICED SDK provide a simple method for an application to integrate Bluetooth Mesh functionality.

Mesh Core library of the WICED SDK provides access to the BLE Mesh Core.

Typically application will not access BLE Mesh Core directly but through the BLE Mesh Models library functions.

uint16_t wiced_bt_core_lower_transport_ack_timeout_ms

Lower Transport Layer ACK timeout.
