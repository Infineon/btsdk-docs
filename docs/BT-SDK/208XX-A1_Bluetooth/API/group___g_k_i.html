void wiced_gki_delay_us ( uint32_t  delay)

This function provides a specified amount of delay in microseconds.

The function overhead is about 18us at 16MHz. It is not possible to get a shorter delay than that. This function can be used before the ARM timers are initialized.

Parameters
delayDelay time in microseconds
