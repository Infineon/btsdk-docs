Detailed Description

Macro Definition Documentation

#define pwmIdToMask (   id)    (1<<id)

Defines the BCM standard pwm driver.

It is compiled and used with BRCM standard applications [Mouse or Keyboard].PWM HW block has 4 PWMs Channels(10 bit) This macros enable to switch from Pwm to its mask value. These mask values are used in enable/disable case.

Enumeration Type Documentation
